--- CelularPOO
Projeto de console em C# desenvolvido como parte do desafio da DIO na trilha .NET, com foco em Programação Orientada a Objetos (POO).

--- Objetivo
Modelar um sistema que simula o funcionamento de celulares de diferentes marcas, utilizando os pilares da POO: abstração, herança, polimorfismo e encapsulamento.

--- estrutura do Projeto
- Smartphone: classe abstrata com atributos e métodos comuns a qualquer celular.
- Nokia e Iphone: classes derivadas que implementam o método InstalarAplicativo de forma específica.
- Program.cs: ponto de entrada do programa, onde os objetos são instanciados e testados.

--- Exemplo de saída
- Ligando...
Instalando "WhatsApp" no Nokia via Nokia Store...
- Recebendo ligação...
Instalando "Instagram" no iPhone via App Store...

-- Tecnologias Utilizadas
.NET 6 (ou superior)
C#
Visual Studio ou VS Code

--- Conceitos Aplicados
- Classes abstratas
- Herança
- Polimorfismo
- Encapsulamento
- Organização modular de código
